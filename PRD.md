# PromptLens PRD - AI 视频创作闭环平台

## 1. 产品概述

### 1.1 产品定位

PromptLens 是一款 AI 视频提示词分析工具，帮助用户：
1. **发现灵感** — 分析热门 AI 视频，反推其创作提示词
2. **二次创作** — 修改提示词，添加个性化元素
3. **生成视频** — 用优化后的提示词生成新视频

目标是打造「**视频分析 → 提示词优化 → 视频生成**」的闭环创作体验。

### 1.2 目标用户

| 用户类型 | 需求 |
|---------|------|
| AI 视频创作者 | 学习爆款视频的提示词，快速复刻创作 |
| 短视频从业者 | 高效分析竞品视频，提取创意元素 |
| AI 爱好者 | 探索 AI 视频生成，玩转各种提示词 |

- 通用 AI 对话助手，类似 ChatGPT
- 支持多轮对话上下文
- 用户可选择 AI 提供商（DeepSeek / 智谱 / OpenRouter）

**用户场景：**
- 问：我这个提示词能生成什么风格的视频？
- 问：帮我优化这个提示词，让动作更流畅
- 问：这个画面适合什么场景的音乐？

**UI 布局：**
```
┌─────────────────────────────────────────┐
│  助手                    [DeepSeek ▼]   │
├─────────────────────────────────────────┤
│                                         │
│  用户: 帮我分析这个画面的风格             │
│                                         │
│  AI: 从画面来看，这是一个赛博朋克风格... │
│                                         │
│  用户: 具体有哪些元素？                   │
│                                         │
│  AI: 主要包括：霓虹灯、未来建筑、机械... │
│                                         │
├─────────────────────────────────────────┤
```

**技术方案：**
- 新增 API 路由：`POST /api/agent-chat`
- 参考 `lib/ai/analyzer.ts` 的 AI 调用模式
- 支持流式输出（SSE）

#### 3.2.2 AI 视频生成（生成 Tab）

**功能描述：**
- 输入提示词文本，调用 Kie.ai (Sora 2) 生成视频
- 支持查询生成状态
- 返回视频 URL 供预览和下载

**用户场景：**
1. 用户在「画面分析」中得到提示词
2. 点击「用此提示词生成视频」按钮
3. 自动跳转到「生成」Tab，提示词已填充
4. 点击生成，等待视频完成
5. 预览并下载生成的视频

**UI 布局：**
```
┌─────────────────────────────────────────┐
│  视频生成                    [Kie.ai ▼]  │
├─────────────────────────────────────────┤
│                                         │
│  提示词输入：                            │
│  ┌─────────────────────────────────────┐│
│  │ 一个赛博朋克风格的未来城市，          ││
│  │ 霓虹灯闪烁，飞车穿梭...              ││
│  └─────────────────────────────────────┘│
│                                         │
│  [时长: 5秒 ▼]  [分辨率: 1080p ▼]       │
│                                         │
│  [           开始生成视频            ]    │
│                                         │
├─────────────────────────────────────────┤
│  生成状态: ✅ 已完成                     │
│  ┌─────────────────────────────────────┐│
│  │        [视频预览播放器]              ││
│  └─────────────────────────────────────┘│
│  [下载视频]  [复制提示词]               │
└─────────────────────────────────────────┘
```

**技术方案：**
- 新增 API 路由：`POST /api/video-generate`
- 调用本地代理（开发环境）：`http://localhost:3000/api/video`
- 轮询状态：`http://localhost:3000/api/status`
- 生产环境：通过环境变量 `KIING_PROXY_URL` 控制

## 4. 用户流程

### 4.1 完整创作闭环

```
1. 登录 → 进入 Dashboard
2. 点击「画面分析」Tab
3. 上传喜欢的 AI 视频
4. AI 分析生成提示词
5. 复制提示词，跳转「生成」Tab
6. 点击生成，等待视频完成
7. 预览/下载视频
8. 发布到平台 → 获得反馈 → 回到 Step 1
```

### 4.2 辅助创作流程

```
1. 登录 → 进入 Dashboard
2. 点击「助手」Tab
3. 粘贴刚生成的提示词
4. 问：帮我优化这个提示词
5. AI 返回优化后的版本
6. 跳转「生成」Tab 生成视频
```

## 5. 技术架构

### 5.1 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Next.js 15 + React 19 + TypeScript | 页面渲染 |
| UI | Tailwind CSS + shadcn/ui | 组件库 |
| 后端 | Next.js API Routes | API 路由 |
| 数据库 | Supabase PostgreSQL + Drizzle ORM | 数据存储 |
| 认证 | better-auth | Google 登录 |
| 存储 | Backblaze B2 | 文件存储 |
| AI 分析 | 智谱AI / Gemini / OpenRouter | 视频分析 |
| AI 生成 | Kie.ai (Sora 2) | 视频生成 |

### 5.2 API 设计

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/agent-chat` | AI 对话（支持流式） |
| POST | `/api/video-generate` | 创建视频生成任务 |
| GET | `/api/video-generate/status` | 查询生成状态 |

### 5.3 数据模型

```typescript
// 视频生成记录
interface VideoGeneration {

  prompt: string;
  duration: number;
  resolution: string;
  status: 'pending' | 'processing' | 'completed' | 'failed';
  videoUrl?: string;
  error?: string;
  createdAt: Date;
  updatedAt: Date;
}
```

## 7. 注意事项

### 7.1 环境限制

| 环境 | 限制 | 解决方案 |
|------|------|----------|
| Vercel 生产 | 无法访问 localhost | 使用环境变量配置代理地址 |

### 7.2 功能开关

